# Dockerized Flask Backend 🚀

A basic Flask backend application built with Python and Flask, running inside a Docker container.

---

## 📌 Technologies Used
- Python
- Flask
- Docker

---

## 📂 Project Structure

```bash
flask-backend/
│
├── app.py
├── Dockerfile
└── requirements.txt
```

---

## ✨ Features
- Simple Flask web application
- Docker container support
- Runs on port 5000
- Easy deployment using Docker

---

## ⚙️ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/MamtaGupta4/dockerized-flask-backend.git
```

### Move to Project Directory

```bash
cd flask-backend
```

---

## 🐳 Run with Docker

### Build Docker Image

```bash
docker build -t flask-backend .
```

### Run Docker Container

```bash
docker run -p 5000:5000 flask-backend
```

---

## 🌐 Application Output

Open browser and visit:

```bash
http://localhost:5000
```

Output:

```bash
Hello Mamta Flask App 🚀
```
